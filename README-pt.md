### Instalação

Clone o projeto

```bash
> git clone https://github.com/Coding-in-community/whatsappbot-project_alice.git
> cd whatsappbot-project_alice
```

Instale as dependências:

```bash
> npm install
```

### Uso

```bash
> node index.js
```

Uma vez que o bot esteja rodando você pode escanear o qrcode
